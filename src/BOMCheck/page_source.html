Part # Starts With: Part numbers will be matched that start with the search term. For example, 'lm339n' will return matching part numbers such as 'LM339N1', 'LM339N-500', etc. Special characters will be excluded if the "Ignore Special Characters" option is checked.

Ignore Special Characters: When selected, any special characters (~ $ ` ' @ % ^ & ( ) - _ = { } \ [ ] : ; < > / | " ? .) will be excluded from the search term and results will be returned with or without these characters. Note: the type ahead feature for the Part # Starts With and Exact Part # options is not available when this option is unchecked.
